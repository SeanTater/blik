@use super::base; @use super::statics::*; @() Blik – Login

Login to access your Blik stories

Can't find the token?

Typically when you start Blik, it will launch your web browser and log you in instantly. That doesn't always work if you install it as a different user, like when you install with pacman or inside docker. In that case, it will print out a code for you to login with, which you enter here. The token only lasts 15 minutes, but if you missed it, just restart Blik to get a new one.

Adding a new device?

Once you have logged in on one device, it will remain logged in for 6 months unless you reset the Blik secret (which logs everyone out). You can use any already logged in device to add another device, so it usually takes just a few seconds.

There are no passwords in Blik. Instead, you get single-use tokens that last 15 minutes. At some point we may add users but mostly it's not necessary because starting a new Blik server only takes a few seconds anyway.
